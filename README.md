# flat-eurorack-busboard

A flat back Eurorack Busboard

406x30mm

free to use copy change manufacture and distribute


BOM

2x 1k 0804 resistor

1x 220 0804 resistor

2x led red 0804

1x led green 0804

17 x 2x8 P 16 pin 2.54 mm SMT Pin Header (or 5 for the small version)
